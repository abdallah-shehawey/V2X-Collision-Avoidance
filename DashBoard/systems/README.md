# V2V System Illustrations — صور شرح أنظمة V2V

صور توضيحية (top-down) لأنظمة الأمان الخمسة اللي بتشتغل بالاتصال بين
السيارات (Vehicle-to-Vehicle)، بنفس ثيم الـ dashboard الأسود. الهدف منها إنك
تشرح كل نظام بالصورة في العرض / المناقشة.

## الأنظمة

| الملف | النظام | بالعربي | القيم من الكود |
|------|--------|---------|----------------|
| `fcw`  | Forward Collision Warning        | تحذير الاصطدام الأمامي            | TTC ≤ 3s تحذير، ≤ 2s حرج |
| `eebl` | Emergency Electronic Brake Light | إضاءة الفرامل الإلكترونية الطارئة | تباطؤ > 4، TTC ≤ 3s / 2s، مدى خلفي 400 سم |
| `bsw`  | Blind Spot Warning               | تحذير النقطة العمياء             | مسافة جانبية < 150 سم |
| `dnpw` | Do Not Pass Warning              | تحذير عدم التجاوز                 | TTC ≤ 6s تحذير، ≤ 4s حرج، مقدمة < 300 سم |
| `ima`  | Intersection Movement Assist     | مساعد الحركة عند التقاطعات        | نطاق 20 م، زمن وصول ≤ 4s / 2s |

## الصيغ

- `*.svg` — متجهية، تكبر لأي مقاس من غير ما تبوّظ (للويب والشرائح الحديثة).
- `png/*.png` — صور عالية الجودة (عرض 2000px) لـ PowerPoint / الطباعة.

## العرض (gallery)

افتح `index.html` في المتصفح — بيعرض الصور الخمسة، وبالضغط على أي صورة تكبر
ملء الشاشة للعرض، والأسهم ← → للتنقل.

## تعديل / إعادة توليد

كل الرسم متولّد من سكربت واحد. عدّل القيم أو الألوان في `generate.py` وبعدين:

```bash
python3 generate.py                       # يولّد ملفات الـ SVG
for f in fcw eebl bsw dnpw ima; do        # يصدّر PNG عالي الجودة
  inkscape --export-type=png --export-width=2000 \
           --export-filename=png/$f.png $f.svg
done
```

## مفتاح الألوان (نفس ثيم الـ dashboard)

- 🔵 أزرق = سيارتك (Host)
- ⚫ أسود بستوب أحمر = السيارة الأخرى / مصدر الخطر
- 🟣 دوائر بنفسجية = بث رسائل V2V (DSRC)
- 🟡 أصفر = تحذير، 🔴 أحمر = خطر حرج
